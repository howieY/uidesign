# 简介

{% hint style="info" %}
UIDesign目前仅提供转为快应用项目代码功能，后续会逐步完善，转为其他类型代码。
{% endhint %}

### 设计理念

1. 布局可视化，直观便捷

2. 自动转换代码，降低开发门槛

3. 支持动态数据绑定，做到真正意义上可视化编辑



### 说明

1. 目前暂提供mac版本包，在版本稳定后出Linux/Windows版
2. 编辑器某些功能依赖全局node环境，可先安装node.js
3. 代码生成需要手动编译，在设计编辑界面选中“编译”功能菜单（快捷键CommandOrCtrl+B）
